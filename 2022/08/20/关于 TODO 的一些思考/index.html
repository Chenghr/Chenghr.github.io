关于 TODO 的一些思考

发表于 2022-08-20 更新于 2022-10-05

断断续续的在生活中增加 TODO 安排挺久了，但是一直没有很好地落实，本篇 blog 之前是暑期想要写的工作记录，但是后续也没坚持下去，就在这“残骸”上写一些关于 TODO 的思考吧。

为什么想要用 TODO

TODO 的本质应该是规划自己未来一段时间要做的事情，合理的使用 TODO 可以让人知道短期的目标在于何处，每天向什么方向努力。

最早的 TODO 习惯应该是高中时期，各种科目的各种作业让人眼花缭乱，因此会买个很小巧的本子，记录没有完成的作业，完成一项作业就划掉一项目，任务清空会有很强的成就感。

后来到了大学，就懒散了，没有规划任务的习惯，后来读研，发现身边的时间管理比较优秀的同学都有安排 TODO 的习惯，另外朋友圈中一位活的让我十分佩服的朋友，也是有每日先整理 TODO 的习惯；以及暑期实习时，mentor 每天来到工位上做的第一件事也是整理规划每日事项。

这些优秀的人都有的习惯，我想必定是个好习惯，因此我想将这个习惯贯彻下去，见贤思齐焉。

另外，整理 TODO 事项本身也是对自己短期未来的一个规划，否则每天的生活可能在无序与浑浑噩噩中度过，一段时间过去了，也就过去了，好像一直忙忙碌碌，但又似乎一无所得。因此在我看来，记录 TODO 是生活的一个见证，完成 TODO 是一种自律的体现，是一种生活态度。

想要让自己变得自律，时间不被浑浑噩噩的消耗掉，是我想要贯彻使用 TODO 的另一个原因。

为什么自己一直坚持不下去

仔细想想，自己一直有在断断续续的使用 TODO，但是一直没有长久的坚持下去的原因大概有以下几点：

  1. 每天的工作流程不够固定。坦白来说，每天工作的开始应该是从回顾昨日的 TODO 和整理今天的 TODO 开始的，但是我的每天工作流程经常开始于打开 bilibili…… 刷的差不多了才开始草草的工作。

    这和使用 TODO 的初衷相违背，也说明了本质上我不是个自律的人，依然想着偷懒。

  2. 每天的 TODO 设置并不合理。我经常习惯性的给自己设置过多的任务，导致设置的 TODO 很难及时完成，从 TODO 事项中没有得到任务清空的正反馈，因此没有坚持下去。

  3. 缺乏外部监督。虽然一个习惯的长期坚持需要的是内在驱动，但是习惯的养成过程中，还是需要一定的外力因素推动的。在 TODO 习惯的前期培养中，我还是缺乏了外在压力来促使我完成这个事情。

想要坚持下去有哪些办法

其实也就是上面原因对应的改进啦，大概也就是以下三点吧：

  1. 每日工作的开始固定为梳理 TODO；
  2. 每日 TODO 任务最多不超过3个，任务尽量使少且可执行的，培养自己清空任务的满足感；
  3. 前期每日打卡，强迫自己坚持下去，定下惩罚任务，强迫自己养成习惯。
  4. 定期总结和整理，获得培养这项任务的长期正反馈。

​

希望自己能在10月中真正养成好这项习惯，并坚持下来！

——壬寅年九月初九
