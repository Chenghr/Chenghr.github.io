深度学习

发表于 2022-08-20 更新于 2022-08-21

常见问答总结。

一、基本概念

深层神经网络难以训练的原因

  • 梯度问题

    梯度消失、梯度爆炸。

  • 权重矩阵退化导致模型有效的自由度减少

    在每个层中只有少量的隐藏单元对不同的输入改变它们的激活值，而大部分隐藏单元对不同的输入都是相同的反应

梯度消失问题是什么？产生的原因？解决？

  1. 现象

    随着模型深度的增加，参数的梯度范数指数级减小。

  2. 产生的原因

    • 激活函数选择不正确（sigmoid，tanh都有边缘饱和效应）；
    • 模型参数初始化不恰当（过小）；
    • 学习率设置过小；
    • 模型架构不合理。

  3. 解决方法

    • 使用 relu 等激活函数；
    • 使用 xaiver 初始化或者 kaming 初始化；
    • 使用 BN 层接在激活函数层后面，保证输入分布一致；
    • LSTM 的结构设计也可以改善 RNN 中的梯度消失问题。

梯度爆炸问题是什么？产生的原因？解决？

  1. 现象

    在深度神经网络或者循环神经网络中，梯度在网络更新过程中不断累积，变成非常大的梯度，导致网络权重值大幅更新，网络不稳定的现象。（主要在循环神经网络中出现）

  2. 产生的原因

    • 梯度连乘

  3. 解决方法

    • LSTM、GRU结构
    • 引入 BN 层结构
    • 使用更合适的激活函数
    • 梯度剪切、权重正则

二、BN (批归一化) 层

归一化或者标准化的原因？

  1. 统一量纲，避免过小的数值被吞噬，过大的数值引发数值问题；
  2. 加快梯度下降的求解速度，增加模型收敛效率；（不同特征上下降的步长考虑的）
  3. 缓解梯度消失或者梯度爆炸问题，避免神经元饱和；（保证激活函数的输入分布不发生大的偏移）

BN 层的作用？

  1. 加快网络的训练和收敛的速度（保证每层的数据分布都一致）
  2. 控制梯度爆炸问题；防止梯度消失问题；
  3. 防止过拟合
    • BN的使用使得一个 minibatch 中所有样本都被关联在了一起，同样一个样本的输出不再仅仅取决于样本的本身，也取决于跟这个样本同属一个batch的其他样本，而每次网络都是随机取batch，一定程度上避免了过拟合。
    • BN 的操作破坏了原来的数据分布，一定程度上缓解了过拟合。

为什么BN层一般用在线性层和卷积层后面，而不是放在非线性单元后?

  1. 非线性单元的输出分布形状会在训练过程中变化，归一化无法消除他的方差偏移；
  2. 全连接和卷积层的输出一般是一个对称、非稀疏的一个分布，更加类似高斯分布，对他们进行归一化会产生更加稳定的分布。

BN 在训练阶段和测试阶段的不同？

  1. 训练阶段：

    训练时使用每一批样本的均值和方差进行计算；

  2. 测试阶段：

    使用全量训练样本的均值和方差进行计算，使用移动平均法求解。

BN 为什么不适用于动态的网络结构和RNN网络？

  1. RNN处理的队列是变长的；
  2. RNN 基于时间状态计算，直接使用 BN 需要保存每个时间状态下的均值和方差，效率低下且占用大量内存。

BN 为什么不适用于强化学习和生成模型？

  1. BN 基于一个 mini-batch 的数据计算均值和方差，不是基于整个数据集计算，等于在进行梯度计算时引入了噪声；
  2. 强化学习和生成模型对与噪声较为敏感。

BN 层的优缺点？

  1. 优点：
    • 减少对于超参数的依赖；（droupout 参数以及 L2 正则项约束参数）
    • 加速模型收敛，缓解梯度消失问题、梯度爆炸问题、提高训练精度；
    • 缓解模型的过拟合问题；
  2. 缺点：
    • 批量统计估算不准确，mini-batch 数目很小时，BN 的误差迅速增加。

三、模型相关设置问题

预训练和微调的作用？

  1. 解决深度神经网络需要大量数据集的问题；
  2. 解决模型训练需要大量时间和资源的问题。

模型正则化的目的是什么？有哪些方法？

  1. 目的

    防止模型过拟合，提高模型的泛化性。

  2. 方法

    • L1正则化、L2正则化；
    • 早停法；（这个注意，老是容易忘）
    • Dropout；(随机舍弃神经元及其链接)
    • Drop Connect；（随机将网络权重的子集置为0）

Dropout 在训练阶段和测试阶段有什么不同？

  1. 训练阶段以 p 的概率丢弃神经元；
  2. 测试阶段每个神经元的输出要乘以 1- p，以保证两个阶段的权重拥有相同的期望。

Dropout 为什么可以缓解过拟合问题？

  1. 减少神经元之间复杂的共线性关系；
  2. 等于多个子网络的堆叠，等价于多个子模型的输出取均值，类似集成学习的场景。

Dropout 可能带来什么问题？

  1. 训练时间会大幅增加，变为之前的 2-3 倍；
  2. 使用 dropout 后代价函数不再被明确定义，每次迭代的时候损失函数值可能不是总体上单调递减；（和上一条对应）

四、激活函数

激活函数的作用是什么？为什么需要非线性激活函数？

  1. 作用（线性和非线性两个角度）
    • 引入非线性因素，增加模型的表达能力，学到更复杂的映射关系；
    • 将当前特征空间通过一定的线性映射转换到另一个空间，让数据更好的被分类。
  2. 非线性激活函数的必要性
    • 只是用线性激活函数的话，模型会退化成线性回归模型；（神经元的计算也是线性的，激活函数也是线性的话，神经网络中只有线性部件）
    • 非线性激活函数可以增加模型的表达能力，有利于模型学习、理解复杂的数据。

常见激活函数分类及特点？

  1. sigmoid 函数

    • 优点：

      输出映射在(0, 1)，单调连续且输出范围有限，优化稳定，求导计算简单；（导数 = f * (1-f) ）

    • 缺点：

      • 需要幂运算，计算成本高；

      • 梯度消失问题较为严重，无法完成深层网络的训练；

      • 不是以 0 为中心，导致收敛速度下降。【原因】

        简单来说，就是输出不能以0为中心，导向最优解的方向必须是之字形（方向一致），优化迭代速度慢。

  2. tanh 函数

    • 优点：
      • 与 sigmoid 函数比，输出均值为 0 ，收敛速度更快，减少迭代次数；
      • 单调连续且输出范围有限，优化稳定，求导计算简单；（导数 = 1 - f * f ）
    • 缺点：
      • 需要幂运算，计算成本高；
      • 同样存在梯度消失，因为在两边一样有趋近于0的情况。

  3. ReLU 函数

    • 优点：
      • 梯度不饱和，收敛速度快，极大的改善了梯度消失问题；
      • 不需要进行指数计算，运算速度快，复杂度低；
      • 稀疏激活性，部分神经元输出为0，减少参数的互相依存关系，缓解过拟合问题的发生。
    • 缺点：
      • 对参数初始化和学习率非常敏感；
      • 存在神经元死亡的现象；
      • 函数的输出均值大于0，偏移现象和神经元死亡会共同影响网络的收敛性。

五、参数初始化

常见的参数初始化方式？

  1. 随机初始化

    权重分布符合正态分布，进行随机初始化。

  2. xaiver 初始化

    保证输入和输出方差一致的初始化。【todo：这里可以想一下为什么】

  3. kaming 初始化

    在 xaiver 的基础上除以2，针对 ReLU 的初始化方法。

  4. 预训练初始化

    在预训练阶段进行参数更新，形成初始值。

参数初始化为 0 或者相同值的问题？

  • 每个神经元的计算都是线性的，多层网络中第二层开始，每一层的输入值都是相同的，即等于一个输出，神经网络的效果退化为线性模型，提取特征的能力较差；

参数初始化的建议？

  1. 初始化为较小的随机数

    • 打破网络对称性的普遍作法；认为每个神经元在一开始都是随机的、独一无二的，所以它们会计算出不同的更新，并将自己整合到整个网络的各个部分。
    • 数字不是越小越好，可能出现梯度消失的情况。

  2. 使用 1/sqrt(n) 校准方差

    可以使用 1/sqrt(n) 校准方差；随着输入量变化更新权重向量，保证神经元最初的输出分布大致相同，经验上提高收敛速率。
