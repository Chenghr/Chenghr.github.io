1.2 python 环境以及安装包的管理

  1. conda 管理包和 pip 管理包的区别

    • conda list 和 pip list 的区别
    • conda install & pip install区别 & 查看当前环境下包的情况

    总结：

    先切换到指定虚拟环境，如果是多个虚拟环境下通用的包，例如 numpy 、pandas 之类的包，可以使用 conda install 安装，如果是当前虚拟环境单独使用的包，则可以使用 pip install 安装；

    另外有些包只能使用 pip 安装，conda 中找不到指定包，还有些包需要底层依赖的，可以使用 pip + wheel文件 安装。
