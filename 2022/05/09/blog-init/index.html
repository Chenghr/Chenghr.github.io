博客搭建记录

发表于 2022-05-09 更新于 2022-05-11 分类于 博客

很早就有写博客的念头，碍于各种原因（主要是懒）拖延到现在。

在我看来，个人博客是一个记录自己的学习过程以及成长过程的地方。虽然有那句话“正经人谁写日记啊”，但是偶尔看看曾经的自己是什么样的，也是挺有趣的一件事情。

很早之前在网上查找学习资料的时候就被 hexo 的 next 主题风格深深的吸引住了，极简的风格，很符合程序员的浪漫，因此这次决定用 hexo + github page 的形式搭建自己的博客。

基础工具

搭建博客之前要先安装两个基础工具：Git 以及 node.js。

1. Git 安装

Git是一款免费、开源的分布式版本控制系统，这里我们主要用于同步博客文件到 github 上，具体安装流程网上有很多教程，这里就不细说了。

2. node.js 安装

需要利用 npm 安装 hexo。

node.js 的具体安装过程也略过了。

初步建站

1. github 相关设置

  • 创建 github 账号

    用于后续配置网页博客，注意 github 账户名和个人博客地址息息相关，创建需慎重。

  • 本地电脑配置 ssh key 到 github 账号

    便于后续博客文件的部署推送。

2. 本地初始化博客文件

​ 在本地指定文件夹下进行 hexo 初始化。

3. 本地博客关联到 github page

  • github 中创建同名仓库 github账户名.github.io
  • 本地修改 blog 文件夹下的配置文件 _config.yml 中的 url 字段和deploy 字段

4. 博客推送

  • 使用 hexo g 生成博客静态文件，再使用 hexo d 将文件推送到 github 上；
  • 浏览器中输入 https://github账户名.github.io 查看个人博客主页。
